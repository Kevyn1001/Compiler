# Compiler Group L Language Documentation 

## Glossary

| Reserved Word | Common Use | Obs |
|--- |--- |--- |
| letter | char | --- |
| bolean | bool | --- |
| // | // | line comment |
| <\!--""--> | /\*\*/ | multi-line comment |


### Examples of comments

```cpp
// Correct

letter c;
c = 'G';

c--;

<!--"
bolean d;
d = verdadeiro;

d--;
show(d);  0 == falso "-->

show(c); // F

```
